Sandbox – Sample projects

Try out sample projects that connect to the buoy's ROS 2 network and services.

Chat with AI

Chat with the LLM (Qwen) via ROS 2 Action /llm/chat with modality: "text". Falls back to HTTP.

AI Image View

Upload an image and ask the vision model (Moondream) to describe it. Uses modality: "image".

Turtle Draw

Control a turtle with Twist messages. Select a controller topic (from Gamepad & Joysticks) to drive and draw.

Live Coding

Write and run ROS 2 JavaScript in the browser. Publish, subscribe, and experiment with roslib.
