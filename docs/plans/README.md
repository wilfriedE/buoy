# Implementation Plans

This folder contains detailed plans for features not yet implemented. Each plan is designed so a developer can build off it later without prior context.

## Plans

| Plan | Description |
|------|--------------|
| [networktables-ros-bridge.md](networktables-ros-bridge.md) | NetworkTables connector for FRC robots – bridge FRC RoboRIO with the ROS network, multi-robot support, Features tab enablement |

## How to use a plan

1. Read the **"How to Build Off This Plan"** section first – it defines implementation order, prerequisites, and patterns.
2. Follow the **Implementation Order** steps in sequence.
3. Use the **Codebase Patterns** table to copy from existing files.
4. Refer to **External References** for APIs and libraries.
5. After implementation, move or archive the plan and update this README.
